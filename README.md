# CryptoReg Europe — Bitcoin & Crypto Regulation Tracker

Interactive map of European Bitcoin and crypto regulation across 51 countries.

**Live site:** https://[your-username].github.io/[your-repo-name]/

## Features
- 6 map views: Regulatory Maturity, Tax Burden, MiCA Status, BTC Reserve Policy, Central Bank Posture, Legal Status
- Hover tooltips and click-through country detail panels
- Cross-hatched countries indicate multiple reserve statuses (Germany, Ukraine, Switzerland)
- 51 countries: EU, EEA, non-EEA, microstates, Balkans
- Orange Wheel brand styling

## Deployment
This is a single self-contained `index.html` file — no build step, no dependencies to install.

### GitHub Pages
1. Create a new repository on GitHub
2. Upload `index.html` to the repository root
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://[username].github.io/[repo-name]/`
